Time Formatting Snippets (v3)

If get the typeof of pubdate, we get "string".

typeof(d.pub_date)

So we can run it through a helper function that will return a date object if pub_date isn't one already:

// FUNCTION
var isDateObject = function(date){
    return typeof(date) === 'object' && date === null ? date : new Date(date)
}

// CALL
isDateObject(d.pub_date)

This return the date object rather than a string:

Tue Jun 04 2013 17:12:00 GMT-0400 (EDT)

We can now run it through the time formatting methods in D3. Setting up a var called format:

var format = d3.time.format('%Y-%B-%d')
	format(isDateObject(d.pub_date))

Gives us: 2013-June-04

var format = d3.time.format('%Y')
	format(isDateObject(d.pub_date))

Gives us: 2013

var format = d3.time.format('%y')
	format(isDateObject(d.pub_date))

Gives us: 13

format = d3.time.format('%A, %B %d, %Y')
	format(isDateObject(d.pub_date))

Gives us: Tuesday, June 04, 2013

Feel free to play with different combinations. You can find the directives here, or in the Time Formatting section the D3 documentation.
