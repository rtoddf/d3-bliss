Charts

2015 Oscar Box Office Grosses

The line chart below shows the weekly grosses of seven of the eight 2015 Oscar nominted best pictures. (American Sniper has been omitted because of extreme outliers.) Click on a movie's name to see its grosses for the first ten weeks after its release.

Bar Chart - US Unemployment Rates

This bar chart tracks the unemployment percentages for all fifty states based on data from the United States Department of Labor Bureau of Statistics. A user can hover over a bar to get that states exact percentage of unemployed, as well as sort the chart by clicking the buttons at the top of the chart.

Population Pie Chart

This donut chart shows the US Census 2010 data for Hawaii. The user can hover over each arc, which will animate, and see a tooltip with race. The percentage is also presented in the center of the chart. More data can be added to the tooltip and the center of the chart.

Vertical Bar Chart - State Population

This bar chart is a variation of the horizontal bar chart. It shows state population in a vertical format. The user can hover over any bar to see the exact population total, as well as sort the chart with the buttons at the top. This chart is using old census data.

Age of Population by State - Multiple Pie

This visualization shows all fifty states with the name of each state in the center of the chart. The user can hover over any arc to see the age range and the total poulation number for each age bracket. The data is currently fake for representational purposes.

Multiline Chart - State Unemployment

This line chart is just to represent a multiline chart that shows data from more than one item. Real data, as well as hover states need to be added.

Bad Drivers

FiveThirtyEight.com makes its data available via its Github repo. This series of charts will represent bad driver and insurance statistics for the US states. This page is still under construction.

Planetary Scatterplot

This visualization shows the planets of our solar system as a scatterplot. The chart presents each planet's distance from the sun, density, and size in relation to the other planets. More data can be added to the tooltip or added as a animatable field on the chart.

Periodic Table

This visualization presents the elements in the periodic table. The user can hover over each element to see it animate. This visualization is still under construction. I want to fix the animations, as well as add more data about each element when the user clicks.

Maps

Map Projections

Most projections provided by D3 are created via d3.geo.projection and are configurable: you can rotate the globe, scale or transform the canvas, etc. These are my practice projections.

US States

A standard map of the United States using the Albers USA projection. Each state has been given a tooltip withe name of the state, as well as its abbreviation. More data can be added to the tooltip if desired.

US with Zoom

This map of the US is constructed in the same manner as the previous map, but adds a "click-to-zoom" feature. The race data is from the USA Today US Census API - 2010.

US Districts

A standard map of the United States using the Albers USA projection. Map data is combined with the 114th congress data to present each states congrssional districts. I will be combining this map with congressional representative data to make the map more interactive.

US Counties

A standard map of the United States using the Albers USA projection. Map data is combined with the county and state topojson data to present all US counties.

Individual States

Topojson provides state data for drawing each state. That data can be combined with county data to present all counties within a state. My plan is to add population, area, and other county-relevent data to the tooltips.

Unemployment Choropleth

A Cloropleth map is a map that uses differences in shading, coloring, or the placing of symbols within predefined areas to indicate the average values of a property or quantity in those areas.

This map shows (past) unemployment data.
